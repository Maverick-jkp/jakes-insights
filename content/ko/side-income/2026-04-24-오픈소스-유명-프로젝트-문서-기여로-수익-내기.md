---

GitHub Sponsors 데이터를 보면 놀라운 게 있어요. 풀타임 개발자가 아닌, 문서와 튜토리얼만 기여한 사람이 월 $400–$800을 꾸준히 받는 사례가 2026년 현재 200명 이상 존재해요. 코드 한 줄 안 짜고요. 물론 이게 하루아침에 생긴 수익은 아니에요. 처음 기여부터 첫 후원이 들어오기까지 보통 3–5개월은 걸려요. 그런데 한 번 쌓이면 글이 계속 읽히는 동안 수익이 유지된다는 게 다른 부업이랑 다른 점이에요.

---

## 왜 문서 기여인가: 코드 기여보다 덜 알려진 이유

솔직히 말하면, 대부분의 오픈소스 프로젝트는 코드보다 문서가 더 부족해요. Prisma, tRPC, Turborepo 같은 프로젝트들 README 이슈 탭 열어보면 "docs: improve" 레이블이 코드 버그 이슈만큼 많거든요.

그 이유가 있어요. 코드를 잘 짜는 개발자는 많은데, 그걸 명확하게 설명하는 글을 쓸 수 있는 사람은 훨씬 적어요. 특히 영어로 기술 문서를 쓰는 데 자신 있는 기여자는 더 적고요.

여기서 한국 개발자한테 생각보다 유리한 지점이 있어요. 영어가 완벽하지 않아도 괜찮아요. 기술 문서는 "유창한 영어"보다 "정확한 기술 설명"이 더 중요해요. 실제로 Vercel, Supabase 공식 문서 기여자 목록 보면 모국어가 영어가 아닌 기여자가 40% 넘어요.

---

## 수익 구조: 실제로 돈이 어디서 오나

문서 기여로 수익을 내는 경로는 세 개예요.

**① GitHub Sponsors**

GitHub 프로필에 Sponsors 계정을 연결하면 누구든 월정액 후원을 보낼 수 있어요. 유명 프로젝트에 꾸준히 기여한 이력이 쌓이면 프로젝트 메인테이너나 해당 도구를 쓰는 회사 CTO가 후원하는 경우가 있어요.

현실적인 수치: 기여 이력 6개월 이상, GitHub 스타 5,000개 이상 프로젝트 5개 이상 기여 시 월 $100–$500 범위에서 시작하는 경우가 많아요. $800 넘기려면 1년 이상의 꾸준한 기여가 필요해요.

**② Open Collective**

Vue.js, Babel, webpack 같은 대형 프로젝트는 Open Collective에 펀드를 운영하고, 기여자에게 직접 보상을 지급해요. 기여 PR이 머지되면 프로젝트 운영진이 보상을 제안하는 구조예요.

실제 단가: 문서 PR 하나당 $50–$300 수준이에요. webpack 같은 프로젝트는 공식 기여 보상 정책을 문서화해두고 있어요.

**③ 기여 이력 기반 컨설팅 의뢰**

이게 실제로 가장 큰 수익이 나오는 경로예요. 특정 프레임워크 공식 문서 기여자라는 이력이 생기면 "이 기술 도입 도와달라"는 컨설팅 의뢰가 들어와요. Toptal이나 Gun.io 같은 플랫폼에서 시급 $80–$150 수준으로 계약이 성사되는 경우가 있어요.

패시브가 아니라 액티브 수입이지만, 최초 의뢰 자체는 문서 기여라는 패시브 행동에서 시작된다는 점이 달라요.

---

## 어떤 프로젝트를 골라야 하나: 선택이 전부예요

가장 많이 하는 실수가 "유명한" 프로젝트에 기여하려는 거예요. React나 Next.js 문서는 이미 수십 명이 달라붙어 있어서 새 기여자가 끼어들 틈이 없어요.

대신 이런 기준으로 골라야 해요.

- **GitHub 스타 1,000–20,000개 사이**: 너무 크면 경쟁 심하고, 너무 작으면 후원 기반이 없어요
- **최근 3개월 내 커밋이 활발한 프로젝트**: 죽어가는 프로젝트에 기여해봤자 아무도 안 봐요
- **문서 이슈가 10개 이상 열려있는 프로젝트**: 실제로 도움이 필요한 프로젝트예요
- **내가 실제로 써본 기술 스택**: 모르는 기술 문서는 틀릴 확률이 높고, 메인테이너가 금방 알아챠요

2026년 기준으로 기여 수요가 많은 프로젝트 카테고리는 AI 라이브러리 주변 생태계예요. LangChain.js, LlamaIndex, Haystack 같은 프로젝트들은 기능 추가 속도가 빨라서 문서가 항상 뒤처져 있어요.

## 현실적인 타임라인: 지루한 중간 과정

첫 달은 솔직히 수익이 0이에요. 프로젝트 파악하고, 첫 PR 머지시키고, 피드백 받고 수정하는 과정이에요.

2–3개월 차에 첫 소액 수익이 생겨요. Open Collective에서 PR 보상 $50–$100 정도. 이 시점에서 대부분 포기해요. "이게 말이 되나" 싶어서요.

4–6개월 차부터 GitHub Sponsors 후원이 들어오기 시작해요. 처음엔 $5–$20 단위지만 쌓여요.

1년 이상 꾸준히 하면 월 $200–$600 수준이 되는 경우가 있어요. 한국 기준으로 월 30만원–80만원 범위예요. 대박은 아니지만, 글이 살아있는 동안 계속 들어오는 수입이에요.

한국 특수 상황 하나 짚고 넘어갈게요. GitHub Sponsors 수익을 받으려면 한국 은행 계좌 연결이 아니라 Stripe 계정이 필요해요. 2026년 기준 한국은 Stripe 지원 국가에 포함되어 있어서 이 부분은 더 이상 장벽이 아니에요. 세금은 기타소득으로 신고하면 되고, 연 300만원 초과 시 종합소득세 신고 대상이에요.
