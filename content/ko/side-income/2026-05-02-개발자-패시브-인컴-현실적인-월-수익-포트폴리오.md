미국 시니어 개발자 기준으로, 패시브 채널 3개를 운영하는 사람의 평균 월 수입이 $1,200 안팎이에요. 그런데 실제로 그 숫자에 도달한 사람이 전체 시도자 중 약 8%라는 데이터가 있어요. 나머지 92%는 중간에 멈추거나, 월 $50 아래에서 포기해요. 왜 이렇게 격차가 클까요. 그리고 한국 개발자는 현실적으로 어디서, 얼마나 기대할 수 있을까요.

---

## 패시브 인컴 채널 4개, 진짜 단가와 타임라인

### Gumroad — 디지털 제품

지금 해외에서 개발자 패시브 수입의 "입문"으로 가장 많이 추천되는 채널이에요. 코드 템플릿, CLI 도구, Notion 대시보드, README 템플릿 같은 걸 올려두면 자는 동안에도 팔려요. 실제로요.

단가 범위는 $9-$79. 가장 많이 팔리는 가격대는 $15-$29예요. Gumroad 수수료는 판매액의 10%예요. 월 20개 팔면 $300, 50개 팔면 약 $675 — 단일 제품 기준이에요.

한국에서 적용할 때 현실적인 벽이 하나 있어요. **영어 마케팅**이에요. Gumroad 자체가 SEO에 약하고, 트위터(X)나 Reddit, Hacker News에서 직접 홍보해야 노출이 생겨요. 한국어로 만들면 시장이 작아지고, 영어로 만들면 마케팅 부담이 커져요. 현실적으로는 영어 제품을 만들고, 처음 한 달은 Reddit r/webdev나 Hacker News "Show HN"에 올리는 게 첫 판매까지 가장 빠른 경로예요.

**타임라인:** 제품 제작 1-2주 + 첫 판매까지 평균 4-6주 + 월 $200 안정화까지 4-6개월.

---

### Udemy / Inflearn — 강의 콘텐츠

강의는 "만들고 나면 끝"처럼 보이지만, 실제로는 3개월마다 업데이트 안 하면 별점이 떨어지고 노출이 줄어요. 그게 현실이에요.

Udemy 기준으로 유기적 판매(Udemy 내부 검색)는 판매가의 37%가 크리에이터 몫이에요. 프로모션 코드로 직접 유입시키면 97%까지 가져갈 수 있어요. 강의 하나 가격이 $12-$19(할인 시)니까, 100개 팔면 약 $400-$1,800 — 유입 방식에 따라 차이가 커요.

Inflearn은 한국 시장이라 경쟁이 치열하지만, 한국어로 만들 수 있어요. 수수료는 20-30%, 강의 가격 평균 2만-5만원. 月 50개 팔면 약 100만-175만원이에요. 실제로 Inflearn 상위 강의들은 누적 수강생 1,000명 이상이 대부분인데, 처음 6개월이 제일 힘들어요.

**타임라인:** 강의 제작 4-8주 + 첫 수익 1-2개월 + 월 50만원 안정화까지 6-9개월.

---

### GitHub Sponsors / Patreon — 오픈소스 후원

솔직히 말할게요. 이건 스타 1,000개 이상 리포지토리가 없으면 월 $50 넘기도 쉽지 않아요. GitHub Sponsors 전체 크리에이터의 중간값이 월 $47이에요. 상위 10%가 월 $500 이상이고요.

그런데 이게 의미없는 채널이냐면 그건 아니에요. 이미 오픈소스를 활발하게 하고 있는 개발자라면, 추가 설정 2-3시간으로 시작할 수 있어요. 기존 활동의 "부산물"로 수입이 생기는 구조예요. 새로 시작하는 오픈소스 프로젝트를 Sponsors 목적으로 만드는 건 비효율적이에요.

**타임라인:** 설정 2-3시간, 하지만 의미있는 수익($200+)까지 12개월 이상.

---

### npm 유료 패키지 / SaaS API — 가장 높은 천장

월 $2,000-$5,000 범위의 패시브 수입을 버는 개발자들이 주로 쓰는 채널이에요. 재사용 가능한 라이브러리를 npm에 올리거나, API 형태로 월정액 구독을 받는 방식이에요.

플랫폼은 Stripe로 결제 붙이고, Lemon Squeezy 같은 걸 써서 해외 부가세 처리를 자동화해요. 한국에서 이 채널이 어려운 이유는 **초기 투자가 크다**는 거예요. 아이디어 검증 + 개발 + 마케팅까지 최소 2-3개월의 집중 작업이 필요하고, 실패 확률도 높아요. 대신 성공하면 천장이 제일 높아요.

**타임라인:** 초기 개발 2-3개월 + 첫 유료 고객까지 1-2개월 + 월 $500 안정화까지 9-12개월.

## 포트폴리오로 묶을 때 달라지는 것

패시브 수입은 채널 하나로는 변동성이 너무 커요. 실제로 안정적인 월 $500-$1,000 구간에 있는 개발자들은 대부분 채널 2-3개를 조합하고 있어요.

현실적인 조합 예시:

- **Gumroad 템플릿 2-3개** → 월 $150-$400
- **Inflearn 강의 1개** → 월 50만-100만원
- **GitHub Sponsors** → 월 $30-$80 (있으면 더하기)

이걸 동시에 시작하려고 하면 다 망해요. 순서가 중요해요. 가장 빠르게 첫 수익이 나오는 채널(Gumroad 또는 Inflearn 강의)을 먼저 안정화한 뒤에 두 번째 채널로 확장하는 게 현실적인 접근이에요.

그리고 "패시브"라는 단어에 속으면 안 돼요. 매달 4-6시간의 유지 작업이 없으면 매출이 떨어져요. 업데이트, 댓글 답변, SNS 포스팅, 가격 테스트 — 이게 전부 들어가요.
