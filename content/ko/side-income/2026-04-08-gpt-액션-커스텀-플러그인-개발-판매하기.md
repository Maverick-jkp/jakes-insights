## GPT 액션이 정확히 뭔지부터 짚고 넘어가요

GPT 플러그인은 2024년 초 OpenAI가 "GPT 액션(Actions)"으로 구조를 바꿨어요. 쉽게 말하면, ChatGPT가 외부 API를 직접 호출할 수 있도록 연결해주는 브릿지 코드예요.

예를 들어 이런 것들이에요:
- Notion API랑 연결해서 "오늘 할 일 정리해줘"라고 하면 자동으로 페이지가 만들어지는 GPT
- Google Calendar 액션을 붙여서 "다음 주 회의 잡아줘"가 실제로 작동하는 GPT
- 쇼피파이 스토어 연동 GPT (재고 조회, 주문 확인 등)

기술 스택은 생각보다 간단해요. OpenAPI 스펙(yaml 또는 json)으로 API를 정의하고, 인증 방식(OAuth2, API Key)을 설정하고, ChatGPT GPT Builder에서 액션으로 등록하면 끝이에요. Python/Node.js로 미들웨어 서버 하나 띄우는 수준이거든요.

## 실제 수익 구조: 어떻게 돈이 되나요?

크게 세 가지 루트가 있어요.

**① 직접 수주 (능동 수입, 시작 2–4주)**

Upwork이나 Fiverr에서 "Custom GPT Action" 또는 "ChatGPT Plugin Developer"로 검색하면 지금 이 순간에도 공고가 올라와 있어요. 2026년 4월 기준 Upwork 시세는 **시간당 $45–$120** 범위인데, 한국 개발자가 영어 프로필 잘 세팅하면 $55–$70 선으로 충분히 수주 가능해요.

프로젝트 규모는 보통 두 가지예요:
- 소형 (단일 API 연동): $300–$800, 2–5일 작업
- 중형 (멀티 API + 인증 + 에러 핸들링): $1,500–$4,000, 1–3주 작업

첫 달에 소형 프로젝트 2–3개를 수주하면 **월 60만원–200만원** 수준이에요. 야근 없이요.

**② Gumroad 패키지 판매 (패시브 수입, 수익까지 3–6개월)**

"Notion + ChatGPT Action Starter Kit" 같은 형태로 패키징해서 Gumroad에 올리는 방식이에요. $29–$79 사이 가격대가 가장 잘 팔려요.

솔직히 말할게요. 처음 올리고 아무것도 안 하면 한 달에 0–2개 팔려요. SEO, 트위터(X) 마케팅, 레딧 r/ChatGPT 커뮤니티 홍보를 꾸준히 해야 **월 $300–$800** 수준이 나오기 시작해요. 빠른 수입이 목표라면 Gumroad는 부업의 부업이에요.

**③ 월정액 SaaS화 (가장 높은 천장, 가장 긴 준비)**

특정 산업군(부동산, 법률, 의료 행정 등)에 특화된 GPT 액션을 만들어서 월 $49–$149에 구독료로 받는 구조예요. 고객 10명이면 월 $500–$1,500. 50명이면 $2,500–$7,500이에요. 단, 여기까지 가려면 6–12개월의 초기 투자 기간이 필요해요.

## 한국에서 실제로 하면 어떤 장벽이 있나요?

**가능한 부분:**
- OpenAI API 접근은 한국에서 완전히 가능해요 (카드 결제, 월 $5–$20으로 시작 가능)
- Upwork 한국 계정 개설 + 달러 수령 가능 (Payoneer 또는 페이팔 연동)
- Gumroad도 한국에서 판매자 등록 가능해요 (페이팔 또는 은행 계좌)

**어려운 부분:**
- 영어 프로필/포트폴리오가 필수예요. 번역기를 써도 되는데, 클라이언트 메시지는 직접 써야 해서 영어 작문 최소 B2 수준이 필요해요.
- 세금 처리가 좀 복잡해요. Upwork 수익은 프리랜서 종합소득세 신고 대상이고, Gumroad는 해외 플랫폼 소득이라 기타소득 또는 사업소득으로 분류돼요. 연 수익 500만원 넘으면 세무사 상담 한 번 받는 게 나아요.
- ChatGPT Store 내 GPT 수익 공유 프로그램은 한국 계정에선 아직 제한적이에요 (2026년 4월 기준 미국/일부 국가 우선). 그래서 ChatGPT Store 직접 판매보다 Upwork + Gumroad 조합이 현실적이에요.

## 지루한 중간 과정 — 진짜 어떻게 되나요?

첫 달: Upwork 프로필 만들고, 샘플 프로젝트 1–2개 만들어서 포트폴리오에 올려요. 실제 제안서를 10–15개 보내는데, 응답률은 10–20% 정도예요. 첫 수주까지 2–4주 걸려요.

두 번째 달: 첫 프로젝트 완료 후 리뷰가 생기면 응답률이 올라가요. 월 60만원–120만원 수준이 생기기 시작해요.

세 번째 달부터: 단골 클라이언트 1–2명이 생기거나, Gumroad 패키지가 팔리기 시작해요. 이 시점에 Upwork + Gumroad를 병행하면 월 150만원–300만원도 가능해요.

핵심은 "플랫폼을 익히는 첫 달이 가장 느리다"는 거예요. 개발 자체보다 제안서 쓰는 방법, 클라이언트 응대하는 방법을 익히는 데 시간이 더 걸려요.
